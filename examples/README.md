# Examples for AshToonEx

This directory contains example code demonstrating how to use AshToonEx.

## Files

- `run_basic.exs` - Basic usage with maps, fragments, and ToonEx.Protocol
- `run_advanced.exs` - Advanced features including Phoenix serializer and complex structures
- `user.ex` - Example User module
- `profile.ex` - Example Profile module

## Running Examples

```bash
# Basic example
mix run examples/run_basic.exs

# Advanced example
mix run examples/run_advanced.exs
```
